Simple Thread is a simple communication application developed using Flutter and Firebase.

## Features

- User authentication: Users can create accounts and log in to the application.
- Real-time messaging: Users can send and receive messages in real-time.
- User profiles: Users can create and edit their profiles.
- Group chat: Users can create and join group chats.
- Notifications: Users can receive notifications for new messages and other events.

## Technologies Used

- Flutter: A mobile application development framework.
- Firebase: A cloud-based platform for building mobile and web applications.
- Dart: A programming language used by Flutter.
- VS Code: A code editor.
- Git: A version control
- GitHub: A cloud-based platform for hosting and collaborating on Git repositories.
- Postman: A tool for testing and debugging APIs.
- Test Magic: A tool for testing and debugging mobile applications.

## Getting Started

To get started with Simple Thread, you will need to have Flutter and Firebase set up on your machine. You can find instructions for setting up Flutter and Firebase in the Flutter and Firebase documentation.
Once you have Flutter and Firebase set up, you can clone the Simple Thread repository from GitHub and run the application using the following command:

```bash
flutter run
```
